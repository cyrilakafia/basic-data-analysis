1. Importing Libraries

First, let's import the necessary libraries.

In[]:
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

  • pandas : Think of pandas as a supercharged spreadsheet that you can control with code. It helps you organize and analyze your data efficiently. You can use pandas to load data from various file formats (CSV, Excel, etc.), perform data cleaning, manipulation, and analysis.

  • numpy: numpy is like a powerful calculator that can handle lots of numbers at once, making it easy to perform complex mathematical calculations on large datasets. numpy is often used for performing mathematical operations on large datasets, such as calculating the mean, median, standard deviation, and more.

  • matplotlib: matplotlib is like an artist's toolkit for drawing graphs and charts, allowing you to visualize your data in various ways to see patterns and insights. You can use matplotlib to create line graphs, scatter plots, bar charts, histograms, and many other types of visualizations to help you understand your data better.

4. Data Visualization with matplotlib

Now, let's create some plots to visualize the data.

Line Graph

Let's create a simple line graph.

In[]:
# Line graph
plt.figure(figsize=(10, 6)) # Set the size of the plot
plt.plot(df['A'], label='A') # Plot the 'A' column
plt.plot(df['B'], label='B') # Plot the 'B' column
plt.xlabel('Index') # Set the x-axis label
plt.ylabel('Value') # Set the y-axis label
plt.title('Line Graph') # Set the title
plt.legend() # Show the legend
plt.show() # Show the plot

Scatter Plot

Next, let's create a scatter plot.

In[]:
# Scatter plot
plt.figure(figsize=(10, 6)) # Set the size of the plot
plt.scatter(df['A'], df['B'])    # Plot the 'A' column against the 'B' column
plt.xlabel('A') # Set the x-axis label
plt.ylabel('B') # Set the y-axis label
plt.title('Scatter Plot')   # Set the title
plt.show()  # Show the plot
In[]:
# Scatter plot
plt.figure(figsize=(10, 6)) # Set the size of the plot
plt.scatter(df['A'], df['B'], label='Type 1')    # Plot the 'A' column against the 'B' column
plt.scatter(df['A'], df['C'], label='Type 2')    # Plot the 'A' column against the 'C' column
plt.legend()  # Show the legend
plt.xlabel('A') # Set the x-axis label
plt.ylabel('B') # Set the y-axis label
plt.title('Scatter Plot')   # Set the title
plt.show()  # Show the plot

Bar Plot (with error bars)

Now, let's create a bar plot with error bars representing the standard error of the mean.

In[]:
# Bar plot with error bars
means = df.mean() # Calculate the mean of each column
errors = df.sem() # Calculate the standard error of the mean of each column
plt.figure(figsize=(10, 6)) # Set the size of the plot
plt.bar(means.index, means.values, yerr=errors.values, capsize=5)   # Plot the mean values with error bars
plt.xlabel('Variable')  # Set the x-axis label
plt.ylabel('Mean Value')    # Set the y-axis label
plt.title('Bar Plot with Error Bars')   # Set the title
plt.show()  # Show the plot

Histogram

Let's create a histogram to visualize the distribution of data.

In[]:
# Histogram
plt.figure(figsize=(10, 6)) # Set the size of the plot
plt.hist(df['A'], bins=20, alpha=0.7, label='A') # Plot the 'A' column
plt.xlabel('Value') # Set the x-axis label
plt.ylabel('Frequency') # Set the y-axis label
plt.title('Histogram')  # Set the title
plt.legend()    # Show the legend
plt.show()  # Show the plot

Subplots

Finally, let's create subplots to display multiple plots side by side.

In[]:
# Subplots
fig, axs = plt.subplots(1, 3, figsize=(18, 6)) # Create a figure and a set of subplots. We want 1 row and 3 columns of subplots

# Line graph
axs[0].plot(df['A'], label='A') # Plot the 'A' column
axs[0].set_title('Line Graph')  # Set the title
axs[0].set_xlabel('Index')  # Set the x-axis label
axs[0].set_ylabel('Value')  # Set the y-axis label
axs[0].legend() # Show the legend

# Scatter plot
axs[1].scatter(df['A'], df['B'], alpha=0.5) # Plot the 'A' column against the 'B' column
axs[1].set_title('Scatter Plot')    # Set the title
axs[1].set_xlabel('A')  # Set the x-axis label
axs[1].set_ylabel('B')  # Set the y-axis label

# Bar plot
axs[2].bar(means.index, means.values, yerr=errors.values, capsize=5)    # Plot the mean values with error bars
axs[2].set_title('Bar Plot with Error Bars')    # Set the title
axs[2].set_xlabel('Variable')   # Set the x-axis label
axs[2].set_ylabel('Mean Value') # Set the y-axis label

plt.tight_layout()  # Adjust the subplots to fit into the figure
plt.show()  # Show the plot
